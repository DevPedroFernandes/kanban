# Kanban de Mentoria

Sistema Kanban desenvolvido durante mentoria profissional para acompanhamento de metas e tarefas. O projeto permite organizar atividades em colunas (To Do, Doing, Done, Backlog), arrastar cards, e exportar/importar dados em JSON.

## Funcionalidades
- ✅ Quadro Kanban interativo com 4 colunas
- ✅ Arrastar e soltar (drag-and-drop) para mover tarefas
- ✅ Armazenamento local (localStorage) – os dados persistem mesmo após fechar o navegador
- ✅ Exportar dados em JSON (para compartilhar com mentor)
- ✅ Importar tarefas (formato JSON)
- ✅ Estatísticas em tempo real (total de tarefas, concluídas, progresso)

## Tecnologias utilizadas
- HTML5
- CSS3 (design responsivo)
- JavaScript (ES6+)
- Font Awesome para ícones

## Como usar
1. Faça o download do arquivo `index.html` (ou clone este repositório).
2. Abra o arquivo no navegador (Chrome, Edge, Firefox).
3. Comece a adicionar suas tarefas!
4. Para exportar seu progresso, clique em "Exportar Progresso" e copie o JSON gerado.
5. Para importar tarefas do mentor, clique em "Importar Tarefas" e cole o JSON recebido.
